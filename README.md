# group3-IASproject-NIDS
Machine Learning-Based Network Intrusion Detection  System (NIDS) with Real-Time Alerting Building a digital security guard that watches all network traffic

Dataset Note: To comply with file size limits, only a small snippet of the dataset (data/sample_dataset.csv) is included in this repository for demonstration purposes. To run the full training pipeline, please download the complete NSL-KDD dataset from [https://www.kaggle.com/datasets/kiranmahesh/nslkdd] and place the kdd_train.csv file in the root directory.
